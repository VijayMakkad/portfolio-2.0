## Overview

IdeaClinic Forum is a full-featured community platform built for the Department of Entrepreneurship and Innovation (DEI) at SRMIST. It gives students and faculty a dedicated space to discuss ideas, share resources, and collaborate on entrepreneurship programs. Built with **Next.js** and **Supabase**, it features real-time updates and secure authentication. 

You can view the source code on GitHub here: [ideaclinic_forum](https://github.com/VijayMakkad/ideaclinic_forum).

The platform supports threaded discussions, categories, user profiles, and real-time updates so participants see new posts and replies instantly without refreshing the page.

## Tech Stack

The application is built with **Next.js** and **TypeScript** for a type-safe, server-rendered frontend with fast page transitions. **Supabase** handles authentication, PostgreSQL database storage, and real-time subscriptions that push live updates to connected clients. Tailwind CSS provides a responsive, accessible UI that works across desktop and mobile devices.

## Highlights

- Delivered a production-ready forum deployed on Vercel with a live URL used by the DEI community.
- Implemented real-time post and reply updates using Supabase subscriptions.
- Built role-aware flows for students, mentors, and administrators within the entrepreneurship program.
- Designed a clean, mobile-first interface that encourages ongoing community engagement.
